# Number Recognizer
Handwritten digit recognizer using keras and pygame, trained with the MNIST dataset.

Install requirements using ```pip install -r requirements.txt```

Controls:
  - R = resets the board
  - Q = quits the program
  - Left click = Draw
  - Right click = Erase
